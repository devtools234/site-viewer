# 🌐 Website Proxy Tool

ابزار نمایش آنلاین وب‌سایت‌های خارجی با استفاده از GitHub Actions و GitHub Pages

## 📋 توضیحات

این پروژه یک GitHub Action است که به شما امکان می‌دهد هر وب‌سایت خارجی را دانلود کرده و به صورت آنلاین روی GitHub Pages نمایش دهید، بدون نیاز به دانلود دستی یا نصب نرم‌افزار.

## 🚀 نحوه استفاده

### مرحله ۱: فعال‌سازی GitHub Pages

1. به **Settings** مخزن بروید
2. از منوی سمت چپ **Pages** را انتخاب کنید
3. در قسمت **Source**، شاخه `gh-pages` را انتخاب کنید
4. روی **Save** کلیک کنید

### مرحله ۲: اجرای Action

1. به تب **Actions** در بالای صفحه مخزن بروید
2. روی **Deploy Website Proxy** کلیک کنید
3. دکمه **Run workflow** را بزنید
4. آدرس سایت مورد نظر را وارد کنید (مثال: `https://example.com`)
5. روی **Run workflow** کلیک کنید
6. منتظر بمانید تا Action اجرا شود (۲-۵ دقیقه)
7. لینک سایت دیپلوی شده در لاگ نمایش داده می‌شود
