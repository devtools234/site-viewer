🌐 Website Proxy Tool

ابزار نمایش آنلاین وب‌سایت‌های خارجی

📋 راهنمای استفاده

  1. به تب Actions در بالای صفحه مخزن GitHub بروید
  2. روی Deploy Website Proxy کلیک کنید
  3. دکمه Run workflow را بزنید
  4. آدرس سایت مورد نظر را وارد کنید (مثال: https://example.com)
  5. منتظر بمانید تا Action اجرا شود (۲-۵ دقیقه)
  6. لینک سایت دیپلوی شده در لاگ نمایش داده می‌شود

⚠️ نکات مهم

  • GitHub Pages باید در تنظیمات مخزن فعال باشد (Settings → Pages → gh-pages branch)
  • برخی سایت‌ها به دلیل CORS یا JavaScript پیچیده ممکن است کامل کار نکنند
  • فقط محتوای استاتیک (HTML, CSS, JS, تصاویر) دانلود می‌شود
  • سایت‌های نیازمند احراز هویت پشتیبانی نمی‌شوند

✨ قابلیت‌ها

  • دانلود خودکار تمام منابع سایت (HTML, CSS, JS, Images)
  • تبدیل لینک‌های نسبی به مطلق
  • دیپلوی خودکار روی GitHub Pages
  • بدون نیاز به نصب نرم‌افزار
  • رایگان و متن‌باز
